GET /users Get all users with pagination
Description

Retrieves a paginated list of users. Returns 10 users per page by default with sorting and filtering capabilities.

Parameters
Name Type Required Description Default
page integer Optional Page number (0-based) 0
size integer Optional Number of items per page 10
sortBy string Optional Field to sort by id
sortDir string Optional Sort direction (asc/desc) asc
city string Optional Filter by city null
keyword string Optional Search in name/email null
Response Schema
data array required
Array of user objects
totalItems integer
Total number of users
currentPage integer
Current page number
Example Response
{ "data": [ { "id": 1, "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "phone": "+1-555-0123", "city": "New York", "createdAt": "2025-08-19T10:30:00Z", "updatedAt": "2025-08-19T10:30:00Z" } ], "totalItems": 25, "totalPages": 3, "currentPage": 0, "pageSize": 10, "hasNext": true, "hasPrevious": false }
POST /users Create a new user
Description

Creates a new user in the system with transaction management and validation.

Request Body Schema
firstName string required
User's first name (2-50 characters)
lastName string required
User's last name (2-50 characters)
email string required
Valid email address (unique)
phone string
Phone number (optional)
city string
City name (optional)
Example Request
{ "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "phone": "+1-555-0123", "city": "New York" }
GET /users/external Get users from external API External
Description

Demonstrates nested API calling by fetching users from JSONPlaceholder external API. This showcases the requirement for external API integration.

Flow: Your Request → Our API → JSONPlaceholder API → Response
External API Details
  • Source: JSONPlaceholder (https://jsonplaceholder.typicode.com)
  • Method: GET /users
  • Timeout: 5 seconds
  • Caching: None (real-time calls)
